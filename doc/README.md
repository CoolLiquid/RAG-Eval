# 知识库测评平台 — 文档中心

面向 **MCP 挂载式知识库接入** 的 RAG 测评 Web 平台。平台不做本地文档入库，通过远程 MCP 服务的指定 Tool 检索知识库，并对检索/RAG 全链路进行自动化测评。

## 产品一句话

> 挂载远程 MCP 知识库 → 用标准题库自动提问 → 调用 MCP 检索 Tool → 评分归因 → 生成可对比的测评报告。

## 核心设计原则

1. **唯一接入方式**：仅支持远程 MCP 服务挂载，不支持本地文档上传。
2. **Tool 入口检索**：平台作为 MCP Client，通过用户指定的检索 Tool 获取知识片段。
3. **职责分离**：知识库存储与索引在 MCP 服务端；测评编排、评分、报告在平台侧。
4. **可观测**：报告必须记录 MCP 调用证据（Tool 名、入参、返回片段、耗时）。
